HEALTH CHECK
============

URL: [http://localhost:5000/api/health](http://localhost:5000/api/health)

Expected Response:
{
"status": "OK",
"message": "CareerCompass API is running"
}

USERS API
=========

POST /api/users/register
URL: [http://localhost:5000/api/users/register](http://localhost:5000/api/users/register)

Body (JSON):
{
"name": "Test User",
"email": "[test@example.com](mailto:test@example.com)",
"passwordHash": "hashedpassword123"
}

Expected Response:
{
"message": "User registered successfully",
"user": {
"id": 3,
"name": "Test User",
"email": "[test@example.com](mailto:test@example.com)",
"stream": null
}
}

---

URL: [http://localhost:5000/api/users](http://localhost:5000/api/users)

"id": 1,

"passwordHash": "hashedpass1",

POSTMAN USAGE TIPS
==================

1. Start backend first
   npm run dev

2. Use raw ‚Üí JSON for POST / PUT

3. Always test /api/health first

4. Status codes to watch:
   200 OK
   400 Bad Request
   404 Not Found
   500 Server Error
